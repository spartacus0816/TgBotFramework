# 安装第三方插件

## 查看

前往 [商店](/store) 即可查看所有发布的插件。

或者使用 `nb-cli` 命令行查看：

```bash
nb plugin list
```

## 安装

前往 [商店](/store) 点击复制 `nb-cli` 安装命令至命令行执行即可安装。

或者自行输入命令安装：

```bash
nb plugin install <plugin-name>
```

或者使用交互模式安装：

```bash
nb plugin install
```

也可以使用 `pip` 安装

```bash
pip install <plugin-name>
```
